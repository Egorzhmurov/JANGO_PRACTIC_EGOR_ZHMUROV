## 🛠️ Початок роботи (Setup Instructions)

Для коректного запуску проекту на вашому комп'ютері (наприклад, на базі **Ryzen 9 7950X**), дотримуйтесь цієї покрокової інструкції:

1. **Підготовка середовища:**
   * Встановіть **Python 3.17** (переконайтеся, що він доданий у PATH).
   * Встановіть IDE **PyCharm**.

2. **Отримання коду через Git:**
   * Запустіть PyCharm.
   * У верхньому меню знайдіть вкладку **VCS (Version Control)**.
   * Оберіть пункт **Get from Version Control...**
   * У вікні, що відкрилося, перейдіть на вкладку **Git**.
   * Вставте посилання на цей репозиторій у поле **URL**.

3. **Вибір місця на диску (Важливо!):**
   * У полі **Directory** натисніть на іконку папки.
   * Оберіть шлях на вашому локальному комп'ютері, куди будуть збережені файли проекту (наприклад, `D:\Projects\DjangoPolls`).
   * Натисніть кнопку **Clone**.

4. **Фінальне налаштування:**
   * Коли проект відкриється, PyCharm автоматично запропонує створити віртуальне середовище. Натисніть **OK**.
   * Відкрийте термінал PyCharm (знизу) та введіть:
     ```bash
     pip install django
     ```
   * Застосуйте міграції та запустіть сервер:
     ```bash
     python manage.py migrate
     python manage.py runserver
     ```
     ## 🛠 Наповнення контентом (Адмін-панель)

Проект підтримує динамічне додавання питань та відповідей через вбудований веб-інтерфейс. Вам не потрібно змінювати код, щоб додати нове опитування.

**Як це працює:**
1. Відкрийте панель керування в браузері: `http://127.0.0.1:8000/admin/`
2. Використовуйте графічний інтерфейс для:
   * **Додавання нових запитань** (Questions).
   * **Створення варіантів відповідей** (Choices) до кожного питання.
   * **Редагування або видалення** існуючих даних.

Після збереження в адмін панелі, нові опитування автоматично з'являться на головній сторінці сайту для користувачів.
